# 🐯 A Tribute to Tigger

A playful single-page web tribute to Tigger, the bouncing tiger from A. A. Milne's *Winnie the Pooh*.

## Overview

A small, self-contained fan page celebrating everything that makes Tigger Tigger — built in pure HTML and CSS with no dependencies or build step.

## Tech stack

Vanilla HTML / CSS — zero dependencies.

```
index.html    # the tribute page
Tigger.jpg    # hero image
favicon.svg   # site icon
```

## Running locally

Open `index.html` in any browser, or serve the folder:

```bash
python -m http.server 8000
# then open http://localhost:8000
```
